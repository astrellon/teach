Adventure Exercises

Alright! Lets aim to make a super simple text adventure game! Don't worry, this doesn't require any fancy. The solution I've made is considered 'complete', in that I've taken many different combinations of game state into account that aren't really necessary.

So first thing we'll need is some basic UI to test with and some extra files. So we'll need:
  • A blank adventure.js file and script tag linking it.
  • A blank adventure.css file and a link tag linking it.
  • A nice big header welcoming the player to adventure.
  • An input for getting the players name.
  • A button for starting the game.
  • A div for displaying all the game world text
  • An input for getting player input.
